🧩 8-Puzzle Problem in Artificial Intelligence 🤖
📖 Problem Statement

The 8-Puzzle Problem is a classic search problem in Artificial Intelligence (AI). It consists of a 3 × 3 grid containing 8 numbered tiles (1–8) and one empty space (□).

🎯 The objective is to move the tiles one at a time into the empty space until the puzzle reaches the desired goal configuration.
.

📜 Rules
🔹 Allowed Moves

The empty space (□) can move:

⬆️ Up
⬇️ Down
⬅️ Left
➡️ Right

🔹 Constraints

❌ Diagonal moves are not allowed.
❌ Only one tile can move at a time.
❌ A tile can move only into the empty space.

🌟 Features of the 8-Puzzle Problem

✨ Single-agent problem
✨ Deterministic environment
✨ Fully observable environment
✨ Sequential decision-making problem
✨ State-space search problem
